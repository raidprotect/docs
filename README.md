# RaidProtect Developer Documentation

This repository contains the internal documentation of RaidProtect. It explains
the internal architecture of the bot and its features.

The documentation is built using [mdBook](https://rust-lang.github.io/mdBook/)
and can be viewed online at https://raidprotect.github.io/docs.

Feel free to open a PR if you want to improve any part of the documentation.
