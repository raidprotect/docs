Discord-Identifikatoren sind einzigartige Nummern, mit denen Sie jeden Benutzer, Kanal, Server oder jede Nachricht auf Discord genau identifizieren können. Sie bieten eine zuverlässige und dauerhafte Möglichkeit, Elemente auf der Plattform zu identifizieren.

:::note Nutzen für die Moderation
Die Verwendung von Discord-Identifikatoren ist besonders nützlich für Moderationsaktionen, da sie Folgendes ermöglicht:
- **Einen abwesenden Benutzer bestrafen**: Jemanden bannen, der den Server verlassen hat
- **Fehler vermeiden**: Die richtige Person auch bei ähnlichen Benutzernamen genau identifizieren
- **Aufzeichnungen führen**: Identifikatoren ändern sich nie, im Gegensatz zu Benutzernamen
- **Konflikte lösen**: Eindeutige Identifizierung bei Sonderzeichen
:::

## Entwicklermodus aktivieren

Sie müssen den Entwicklermodus aktivieren, um Discord-Identifikatoren verwenden zu können.

<SeparatedBox>
<Tabs>
  <TabItem value="desktop" label="Auf Discord Desktop/Web" default>

    1. **Benutzereinstellungen öffnen**
    - Klicken Sie auf das Zahnradsymbol unten links in der Oberfläche

    2. **Erweiterte Einstellungen aufrufen**
    - Navigieren Sie in der Seitenleiste zum Tab **Erweitert**

    3. **Entwicklermodus aktivieren**
    - Aktivieren Sie die Option **"Entwicklermodus"**, indem Sie auf den Kippschalter klicken
    - Die Schaltfläche sollte nach der Aktivierung blau werden

  </TabItem>
  <TabItem value="mobile" label="Auf Discord Mobile">

    1. **Benutzereinstellungen öffnen**
    - Tippen Sie unten rechts auf Ihren Avatar
    - Tippen Sie oben rechts auf das Zahnradsymbol

    2. **Erweiterte Einstellungen aufrufen**
    - Scrollen Sie nach unten und tippen Sie auf **"Erweitert"**

    3. **Entwicklermodus aktivieren**
    - Aktivieren Sie die Option **"Entwicklermodus"**

  </TabItem>
</Tabs>
</SeparatedBox>

## Einen Identifikator kopieren

Sobald der Entwicklermodus aktiviert ist, können Sie den Identifikator jedes Elements kopieren:

<SeparatedBox>
<Tabs>
  <TabItem value="user" label="Benutzer-ID" default>

    - **Rechtsklick** auf den Avatar oder Benutzernamen
    - Wählen Sie **"Benutzer-ID kopieren"**

  </TabItem>
  <TabItem value="channel" label="Kanal-ID">

    - **Rechtsklick** auf den Kanalnamen in der Kanalliste
    - Wählen Sie **"Kanal-ID kopieren"**

  </TabItem>
  <TabItem value="server" label="Server-ID">

    - **Rechtsklick** auf den Servernamen oben links
    - Wählen Sie **"Server-ID kopieren"**

  </TabItem>
  <TabItem value="message" label="Nachrichten-ID">

    - **Rechtsklick** auf eine Nachricht
    - Wählen Sie **"Nachrichten-ID kopieren"**

  </TabItem>
</Tabs>
</SeparatedBox>

## Verwendung mit RaidProtect

Identifikatoren können in RaidProtect-Befehlen anstelle von Erwähnungen verwendet werden:

- [`/ban user:123456789012345678 reason:Violation des règles du serveur`](../features/moderation#ban)
- [`/clear amount:100 user:123456789012345678`](../features/moderation#clear)
- [`/userinfo user:123456789012345678`](../features/utilities#userinfo)
- [`/bypass minage user:123456789012345678`](../features/raid-mode#minage)

:::info
Sie können einen Benutzer anhand seiner Discord-ID bannen, selbst wenn er nicht online oder auf Ihrem Server anwesend ist.
:::
