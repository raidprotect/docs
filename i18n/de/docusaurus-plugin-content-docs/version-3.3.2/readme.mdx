Diese Dokumentation soll erklären, wie RaidProtect funktioniert. Senden Sie uns gerne Ihr Feedback, damit wir sie verbessern können.
Vergessen Sie nicht, [unserem Discord-Server beizutreten](https://raidprotect.bot/discord), um über die neuesten Neuigkeiten des Projekts informiert zu bleiben. Sie können dort auch unser Support-Team erreichen, das Ihre Fragen zur Verwendung des Bots beantwortet. 😄

## 🚀 Erste Schritte mit RaidProtect {#start}

Um RaidProtect zu verwenden, laden Sie ihn einfach mit folgendem Link auf Ihren Server ein.

<DocCard item={{ type: 'link', href: 'https://raidprotect.bot/invite', label: 'RaidProtect einladen' }} />
<br />

:::danger
Achtung, **Kopien des Bots** sind im Umlauf, die **den Namen RaidProtect verwenden**. Laden Sie ihn nur über unsere offizielle Website ein.
:::

### Voraussetzungen {#requirement}

Um das ordnungsgemäße Funktionieren von RaidProtect sicherzustellen:
1. **Erteilen Sie RaidProtect die Administrator-Berechtigung**. Dadurch kann der Bot automatisch alle Ihre Kanäle moderieren.
2. **Platzieren Sie seine Rolle auf derselben Ebene wie die Ihrer Server-Administratoren**. Das Berechtigungssystem erlaubt RaidProtect nicht, Nutzer mit einer höheren Rolle zu sanktionieren.
