## 3.3.1 (09/02/2026) {#3-3-1}

- [Internationalization](./language.md) (addition of German, Spanish, and Portuguese).
- New **[Jail](./features/sanctions.mdx#jail)** system.
  - New [`/jail` command](./features/moderation.mdx#jail) to jail a member.
  - New [`/tempjail` command](./features/moderation.mdx#tempjail) to temporarily jail a member.
  - New [`/unjail` command](./features/moderation.mdx#unjail) to release a member.
  - Configuration of the Jail role and information channel via `/settings`.
- New **[Role-based Mute](./features/sanctions.mdx#mute)** system: beyond a [configurable threshold](./features/sanctions.mdx#mute-threshold), the bot assigns a Mute role instead of the Discord timeout.
  - The `/timeout` and `/untimeout` commands become [`/mute`](./features/moderation.mdx#timeout) and [`/unmute`](./features/moderation.mdx#untimeout). Discord timeout is still used by default below the threshold.
  - New `[force_timeout]` parameter to force Discord timeout regardless of the duration.
- [Sanction via context menu](./features/moderation.mdx#context-menu) (right-click on a profile).
- New [`/my-sanctions` command](./features/utilities.mdx#my-sanctions) to allow members to view their own sanctions.
- New [Sanctions Privacy](./features/sanctions.mdx#sanctions-privacy) setting to control access to personal sanctions.
- New [Show Media](./features/sanctions.mdx#show-medias) setting in sanction reasons.
- Visual redesign of filters in [`/sanctions search`](./features/sanctions.mdx#search) and addition of the [moderator type](./features/sanctions.mdx#search) filter.
- [Import / Export sanctions](./features/sanctions.mdx#import-export) via the support bot.
- **[Information Panels](./features/display.mdx)** system (`/display public`).
- New **[member blocking](./features/utilities.mdx#block)** system for certain RaidProtect features on a server.
  - New [`/block add` command](./features/utilities.mdx#block-add).
  - New [`/block remove` command](./features/utilities.mdx#block-remove).
  - New [`/block list` command](./features/utilities.mdx#block-list).
- New [`/prune` command](./features/utilities.mdx#prune) to kick inactive members with more precise conditions. <SuggestionLink id="256" />
- New [`/channel clear` command](./features/utilities.mdx#channel-clear) to delete and recreate a channel. <SuggestionLink id="543" />
- Logs for `/bypass` commands. <SuggestionLink id="546" />
- New **Anti-Scam** module.
- Fixes:
  - Improved captcha management.
  - Fixed prefix validation.
  - Translation fixes.
- Performance and stability:
  - New translation framework ([`@fca.gg/hermes` v1.2.1](https://github.com/FCAgreatgoals/hermes)).

## 3.3.0 (19/12/2025) {#3-3-0}

- New feature **[Sanctions History](./features/sanctions.mdx)**.
  - New [command `/sanctions search`](./features/sanctions.mdx#search).
  - New [command `/sanctions info`](./features/sanctions.mdx#info).
  - New [command `/sanctions edit`](./features/sanctions.mdx#edit).
  - New [command `/sanctions delete`](./features/sanctions.mdx#delete).
  - Automod sanctions are added to the sanctions history.
  - Bans, kicks and timeouts are added to history and logged. <SuggestionLink id="151" />
  - Acknowledgment receipt of the sanction notification private message.
- New logs for sanctions.
- Ability to configure a dedicated sanctions log channel.
- New [command `/warn`](./features/moderation.mdx#warn). <SuggestionLink id="52" />
- New [command `/untimeout`](./features/moderation.mdx#untimeout). <SuggestionLink id="52" />
- New [command `/tempban`](./features/moderation.mdx#tempban). <SuggestionLink id="524" />
- [RaidMode](./features/raid-mode.md#raid-mode) and [Auto RaidMode](./features/raid-mode.md#duration) automatically deactivate after a certain time. <SuggestionLink id="37" />
- New [duration] parameter for the [command `/raidmode`](./features/raid-mode.md#raid-mode). <SuggestionLink id="37" />
- New [command `/joinlock`](./features/join-lock.mdx) to close invites indefinitely.
- New [command `/channel duplicate`](./features/utilities.mdx#channel-duplicate) to strictly duplicate a channel. <SuggestionLink id="225" />
- New command `/changelog` and buttons in `/settings` and `/about` to view the changelog.
- Acknowledgment receipt of the [Minimum Age](./features/raid-mode.md#minage) private message.
- Fixes:
  - Improved cooldown handling.
  - Fixed various translation errors.
  - Increased maximum sanction reason size (512).
- Performance and stability:
  - Optimized message cache.
  - Optimized database storage.
  - Sanction expiration handling (`@fca.gg/midway` v1.3.2).
  - Revamped emoji management ([`@fca.gg/glyph` v1.1.1](https://github.com/FCAgreatgoals/glyph)).
