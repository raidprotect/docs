## 🚀 Primeros pasos con RaidProtect {#start}

Para empezar a usar RaidProtect, simplemente **invítalo a tu servidor** usando el siguiente enlace. Ten cuidado con las copias: invítalo únicamente desde nuestro sitio web oficial.

:::note
Para garantizar el correcto funcionamiento, asegúrate de otorgarle a RaidProtect el permiso de **Administrador**. Esto le permite moderar automáticamente todos tus canales.

Además, asegúrate de **colocar su rol al mismo nivel que los administradores de tu servidor** (ver la imagen a continuación). El sistema de permisos impide que RaidProtect actúe contra usuarios con un rol superior.

![Captura de pantalla del rol](../../../en/docusaurus-plugin-content-docs/version-3.0.0/assets/role-raidprotect.png)
:::

¡Listo, **RaidProtect ya está en tu servidor de Discord**! Se ha creado un canal llamado `#raidprotect-logs`. No elimines este canal, es esencial para el funcionamiento del bot. Si ya tienes un canal de logs, puedes cambiar el canal de logs de RaidProtect con el comando `?settings logs #nuevo-canal`.

:::note
**Si el canal de logs no se creó automáticamente**, deberás configurar manualmente el bot ejecutando el comando `?setup` en tu servidor.
:::

## 👥 Acerca del proyecto {#about}

RaidProtect es un proyecto gratuito y comunitario que comenzó en julio de 2018. Creado por [baptiste0928](https://baptiste0928.net/), el proyecto rápidamente ganó una gran base de usuarios y ahora cuenta con un equipo de alrededor de quince voluntarios, especialmente para el soporte a usuarios.
