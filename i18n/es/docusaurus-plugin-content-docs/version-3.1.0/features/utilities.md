## ðŸ‘¤ Informacion de usuario {#userinfo}

El comando `/userinfo` te permite obtener informacion detallada sobre un usuario, como su **fecha de creacion de cuenta de Discord** y la **fecha en que se unio** a tu servidor (si es miembro).

Usa el comando: ```/userinfo [@usuario]```

Reemplaza `[@usuario]` con la mencion o ID deseado.

## ðŸ§¹ Eliminar un grupo de mensajes {#clear}

El comando `/clear` te permite eliminar rapidamente una cierta cantidad de mensajes en un canal de texto. Puedes especificar un usuario para eliminar solo sus mensajes.

Usa el comando: ```/clear [numero] (@usuario)```

- Reemplaza `[numero]` con la cantidad de mensajes que deseas eliminar (maximo 100).
- Agrega `(@usuario)` usando la mencion o ID para apuntar solo a sus mensajes en el canal.
