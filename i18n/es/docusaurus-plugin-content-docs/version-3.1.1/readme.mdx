### Requisitos {#requirement}

Para asegurar el correcto funcionamiento de RaidProtect:
1. **Otorga el permiso de Administrador** a RaidProtect. Esto le permite moderar automaticamente todos tus canales.
2. **Coloca su rol al mismo nivel que los administradores de tu servidor**. El sistema de permisos no permite que RaidProtect sancione a usuarios con un rol superior al suyo.
