## üë§ Informacion del usuario {#userinfo}

El comando `/userinfo` te permite obtener informacion detallada sobre un usuario.

Usa el comando: ```/userinfo [@user]```

Reemplaza `[@user]` con la mencion o ID deseada.

### üìã Informacion mostrada {#displayed-userinfo}

- **Fecha de creacion de la cuenta de Discord**
- **Foto de perfil** del usuario
- **Banner de perfil**
- **Insignias de perfil**
  - Las insignias de Nitro, Booster, Quest y Originally Name no se muestran.


### üé≠ Informacion de un miembro del servidor {#displayed-memberinfo}

Si el objetivo es un miembro del servidor, la informacion adicional incluye:

- **Fecha de ingreso al servidor**
- **Apodo en el servidor**
- **Numero de roles** y **lista de los primeros 6 roles**
- **Categoria de permisos** (visible solo para moderadores):
