## üìã Duplicar un canal {#channel-duplicate}

El comando `/channel duplicate` te permite duplicar completamente un canal, incluyendo todas sus configuraciones, permisos y ajustes. Va mas alla de la funcion nativa de Discord, que no permite duplicar **categorias** ni **ciertas configuraciones especificas** como las etiquetas de canales de foro.

Usa el comando: ```/channel duplicate [channel]```

Reemplaza `[channel]` con la mencion o el ID del canal a duplicar.

### üéØ Caracteristicas {#duplicate-features}

Este comando duplica todos los siguientes elementos:

- **Nombre del canal**
- **Permisos** (roles y usuarios)
- **Tipo de canal** (texto, voz, foro, etc.)
- **Configuraciones especificas** segun el tipo de canal
- **Posicion** dentro de la categoria (si aplica)

### ‚öôÔ∏è Configuraciones de RaidProtect {#duplicate-config}

Cuando duplicas un canal que tiene configuraciones internas de RaidProtect, el bot te ofrecera automaticamente la opcion de:

- **Reemplazar** la configuracion existente para usar el nuevo canal
- **Duplicar** la configuracion para mantener ambos canales configurados

**Ejemplo**: Si duplicas el canal de registros, RaidProtect te preguntara si deseas actualizar su configuracion para que el nuevo canal se convierta en el canal de registros, o mantener ambos canales configurados segun tu preferencia.

## üë§ Informacion de usuario {#userinfo}

El comando `/userinfo` te permite obtener informacion detallada sobre un usuario.

Usa el comando: ```/userinfo [user]```

Reemplaza `[user]` con la mencion o ID deseado.

:::info
El comando `userinfo` tambien esta [disponible con prefijo](../guides/prefix.md).
:::

### üìã Informacion mostrada {#displayed-userinfo}

- **Fecha de creacion de la cuenta de Discord**
- **Foto de perfil** del usuario
- **Banner del perfil**
- **Insignias del perfil**
  - Las insignias de Nitro, Booster, Quest y Originally Name no se muestran.

## ‚öñÔ∏è Sanciones de Discord {#discord-sanctions}
