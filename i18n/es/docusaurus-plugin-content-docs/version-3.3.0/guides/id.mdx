Los IDs de Discord son numeros unicos que identifican con precision cada usuario, canal, servidor o mensaje en Discord. Son una forma confiable y permanente de identificar elementos en la plataforma.

:::note Relevancia para la moderacion
Usar los IDs de Discord es especialmente util para acciones de moderacion porque te permite:
- **Sancionar a un usuario ausente**: Banear a alguien que ha dejado el servidor
- **Evitar errores**: Identificar con precision a la persona correcta incluso con nombres de usuario similares
- **Mantener un registro**: Los IDs nunca cambian, a diferencia de los nombres de usuario
- **Resolver conflictos**: Identificacion clara en caso de caracteres especiales
:::

## Activar el modo desarrollador

Necesitas activar el modo desarrollador para usar los IDs de Discord.

<SeparatedBox>
<Tabs>
  <TabItem value="desktop" label="En Discord Escritorio/Web" default>

    1. **Abrir configuracion de usuario**
    - Haz clic en el icono de engranaje ⚙️ en la parte inferior izquierda de la interfaz

    2. **Acceder a la configuracion avanzada**
    - En la barra lateral, ve a la pestana **Avanzado**

    3. **Activar el modo desarrollador**
    - Activa la opcion **"Modo desarrollador"** haciendo clic en el boton de alternancia
    - El boton deberia ponerse azul una vez activado

  </TabItem>
  <TabItem value="mobile" label="En Discord Movil">

    1. **Abrir configuracion de usuario**
    - Toca tu avatar en la parte inferior derecha
    - Toca el icono de engranaje ⚙️ en la parte superior derecha de la interfaz

    2. **Acceder a la configuracion avanzada**
    - Desplazate hacia abajo y toca **"Avanzado"**

    3. **Activar el modo desarrollador**
    - Activa la opcion **"Modo desarrollador"**

  </TabItem>
</Tabs>
</SeparatedBox>

## Copiar un ID

Una vez activado el modo desarrollador, puedes copiar el ID de cualquier elemento:

<SeparatedBox>
<Tabs>
  <TabItem value="user" label="ID de usuario" default>

    - **Clic derecho** en el avatar o nombre de usuario
    - Selecciona **"Copiar ID de usuario"**

  </TabItem>
  <TabItem value="channel" label="ID de canal">

    - **Clic derecho** en el nombre del canal en la lista de canales
    - Selecciona **"Copiar ID de canal"**

  </TabItem>
  <TabItem value="server" label="ID de servidor">

    - **Clic derecho** en el nombre del servidor en la parte superior izquierda
    - Selecciona **"Copiar ID de servidor"**

  </TabItem>
  <TabItem value="message" label="ID de mensaje">

    - **Clic derecho** en un mensaje
    - Selecciona **"Copiar ID de mensaje"**

  </TabItem>
</Tabs>
</SeparatedBox>

## Uso con RaidProtect

Los IDs pueden usarse en los comandos de RaidProtect en lugar de menciones:

- [`/ban user:123456789012345678 reason:Violacion de reglas del servidor`](../features/moderation#ban)
- [`/clear amount:100 user:123456789012345678`](../features/moderation#clear)
- [`/userinfo user:123456789012345678`](../features/utilities#userinfo)
- [`/bypass mining user:123456789012345678`](../features/raid-mode#minage)

:::info
Puedes banear a un usuario usando su ID de Discord, incluso si no esta conectado o presente en tu servidor.
:::
