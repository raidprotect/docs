## 🚀 Começar a utilizar o RaidProtect {#start}

Para começar a utilizar o RaidProtect, basta **convidá-lo para o seu servidor** através da seguinte ligação. Tenha cuidado com cópias — convide-o apenas a partir do nosso site oficial.

:::note
Para garantir o funcionamento correto, certifique-se de que concede ao RaidProtect a permissão de **Administrador**. Isto permite-lhe moderar automaticamente todos os seus canais.

Além disso, certifique-se de que **coloca o seu cargo ao mesmo nível dos administradores do seu servidor** (ver a imagem abaixo). O sistema de permissões impede o RaidProtect de agir contra utilizadores com um cargo superior.

![Captura de ecrã do cargo](../../../en/docusaurus-plugin-content-docs/version-3.0.0/assets/role-raidprotect.png)
:::

Pronto, **o RaidProtect já está no seu servidor Discord**! Foi criado um canal chamado `#raidprotect-logs`. Não elimine este canal — é essencial para o funcionamento do bot. Se já tiver um canal de logs, pode alterar o canal de logs do RaidProtect com o comando `?settings logs #novo-canal`.

:::note
**Se o canal de logs não foi criado automaticamente**, terá de configurar manualmente o bot executando o comando `?setup` no seu servidor.
:::

## 👥 Sobre o projeto {#about}

O RaidProtect é um projeto gratuito e comunitário que teve início em julho de 2018. Criado por [baptiste0928](https://baptiste0928.net/), o projeto rapidamente ganhou uma grande base de utilizadores e conta agora com uma equipa de cerca de quinze voluntários, especialmente para o suporte aos utilizadores.
