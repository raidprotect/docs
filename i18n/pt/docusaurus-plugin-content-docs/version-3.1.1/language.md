---
title: Idioma
---

O RaidProtect permite-te escolher o idioma utilizado pelo bot para melhor se adequar a tua comunidade Discord.

:::note
Se o teu servidor estiver configurado como servidor de comunidade (definicao do Discord), o RaidProtect utilizara por defeito o idioma configurado nas **definicoes de comunidade** do servidor.
:::

**Mensagens Publicas:** O idioma configurado afeta apenas as mensagens publicas enviadas pelo RaidProtect no teu servidor (registos, mensagens de captcha, denuncias, etc.).

**Mensagens Efemeras:** Estas mensagens privadas ou temporarias sao apresentadas no idioma do utilizador que interage com o bot.
