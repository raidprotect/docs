### Requisitos {#requirement}

Para garantir o bom funcionamento do RaidProtect:
1. **Concede a permissao de Administrador** ao RaidProtect. Isto permite-lhe moderar automaticamente todos os teus canais.
2. **Coloca o seu cargo ao mesmo nivel que os administradores do teu servidor**. O sistema de permissoes nao permite que o RaidProtect sancione utilizadores com um cargo superior ao dele.
