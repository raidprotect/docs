---
title: Idioma
---

O RaidProtect permite-te escolher o idioma utilizado pelo bot para se adaptar melhor a tua comunidade Discord.

:::note
Se o teu servidor estiver configurado como servidor de comunidade (definicao do Discord), o RaidProtect utilizara por predefinicao o idioma configurado nas **definicoes de comunidade** do servidor.
:::

**Mensagens publicas:** O idioma configurado afeta apenas as mensagens publicas enviadas pelo RaidProtect no teu servidor (registos, mensagens de captcha, denuncias, etc.).

**Mensagens efemeras:** Estas mensagens privadas ou temporarias continuam a ser apresentadas no idioma do utilizador que interage com o bot.
