### Requisitos {#requirement}

Para garantir o correto funcionamento do RaidProtect:
1. **Concede a permissao de Administrador** ao RaidProtect. Isto permite-lhe moderar automaticamente todos os teus canais.
2. **Coloca o seu cargo ao mesmo nivel dos administradores do teu servidor**. O sistema de permissoes nao permite que o RaidProtect sancione utilizadores com um cargo superior ao dele.
