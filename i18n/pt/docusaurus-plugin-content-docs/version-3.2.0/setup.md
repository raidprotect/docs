O RaidProtect simplifica a gestao do servidor com duas ferramentas poderosas: o comando [`/setup`](#install) para uma configuracao guiada passo a passo e o comando [`/settings`](#settings) para ajustar as tuas definicoes a qualquer momento atraves de um menu centralizado. Este guia de instalacao explica como utiliza-los de forma eficaz.

## Instalacao guiada {#install}

O comando `/setup` foi concebido para te ajudar a configurar o RaidProtect de forma rapida ou atraves de uma abordagem detalhada, conforme as tuas necessidades. Oferece dois modos de configuracao: [recomendada](#recommended) ou [avancada](#advanced).

### üîß Configuracao recomendada {#recommended}

Permite-te ativar ou desativar as funcionalidades principais de forma rapida usando um menu de selecao interativo.

1. Usa o comando `/setup`.
2. Seleciona o botao "**Recommended Configuration**".
3. Ativa ou desativa as funcionalidades desejadas usando o menu de selecao.

O bot enviara um resumo das funcionalidades ativadas e das alteracoes que fara no servidor.

## Modificar a configuracao {#settings}

O comando `/settings` e o comando ideal para gerir as tuas definicoes apos a instalacao. Permite-te ver, ajustar ou personalizar as funcionalidades do RaidProtect a qualquer momento, de forma simples e rapida.

### üîç Menu de definicoes {#menu}

1. Escreve `/settings` num canal onde o bot esteja ativo.
2. Navega facilmente entre as diferentes seccoes para encontrar a definicao que desejas modificar.
3. Ajusta as opcoes: Cada categoria apresenta uma lista de opcoes personalizaveis sob a forma de botoes ou menus suspensos.

![Captura de ecra das definicoes](../../../en/docusaurus-plugin-content-docs/current/assets/rp-settings.webp)

### üîÑ Repor uma definicao {#reset}

1. Navega ate a definicao desejada.
2. Clica em "**Reset**".

![Captura de ecra do botao repor](../../../en/docusaurus-plugin-content-docs/current/assets/rp-button-reset.webp)

O bot confirmara a reposicao antes de aplicar as alteracoes.
