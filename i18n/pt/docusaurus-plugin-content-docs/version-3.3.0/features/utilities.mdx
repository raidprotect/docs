## üìã Duplicar um Canal {#channel-duplicate}

O comando `/channel duplicate` permite-lhe duplicar completamente um canal, incluindo todas as suas definicoes, permissoes e configuracoes. Vai alem da funcionalidade nativa do Discord, que nao permite duplicar **categorias** e **certas definicoes especificas** como as tags de canais de forum.

Utilize o comando: ```/channel duplicate [channel]```

Substitua `[channel]` pela mencao ou pelo ID do canal a duplicar.

### üéØ Funcionalidades {#duplicate-features}

Este comando duplica todos os seguintes elementos:

- **Nome do canal**
- **Permissoes** (cargos e utilizadores)
- **Tipo de canal** (texto, voz, forum, etc.)
- **Definicoes especificas** dependendo do tipo de canal
- **Posicao** dentro da categoria (se aplicavel)

### ‚öôÔ∏è Configuracoes do RaidProtect {#duplicate-config}

Quando duplica um canal que tem configuracoes internas do RaidProtect, o bot oferece-lhe automaticamente a opcao de:

- **Substituir** a configuracao existente para utilizar o novo canal
- **Duplicar** a configuracao para manter ambos os canais configurados

**Exemplo**: Se duplicar o canal de registos, o RaidProtect perguntara se pretende atualizar a sua configuracao para que o novo canal se torne o canal de registos, ou manter ambos os canais configurados conforme a sua preferencia.

## üë§ Informacoes do Utilizador {#userinfo}

O comando `/userinfo` permite-lhe obter informacoes detalhadas sobre um utilizador.

Utilize o comando: ```/userinfo [user]```

Substitua `[user]` pela mencao ou ID desejado.

:::info
O comando `userinfo` tambem esta [disponivel com prefixo](../guides/prefix.md).
:::

### üìã Informacoes Apresentadas {#displayed-userinfo}

- **Data de criacao da conta Discord**
- **Foto de perfil** do utilizador
- **Banner do perfil**
- **Distintivos do perfil**
  - Os distintivos Nitro, Booster, Quest e Originally Name nao sao apresentados.

## ‚öñÔ∏è Sancoes do Discord {#discord-sanctions}

Utilize o comando: ```/ds (user)```

Substitua `(user)` pela mencao ou ID desejado.
