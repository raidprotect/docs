### Requisitos {#requirement}

Para garantir o bom funcionamento do RaidProtect:
1. **Conceda a permissao de Administrador** ao RaidProtect. Isto permite-lhe moderar automaticamente todos os seus canais.
2. **Coloque o seu cargo ao mesmo nivel dos administradores do servidor**. O sistema de permissoes nao permite ao RaidProtect sancionar utilizadores com um cargo acima do seu.
