## 🚀 Começar a utilizar o RaidProtect {#start}

Para começar a utilizar o RaidProtect, basta convidá-lo para o seu servidor através do seguinte link.

<DocCard item={{ type: 'link', href: 'https://raidprotect.bot/invite', label: 'Convidar o RaidProtect' }} />
<br />

:::danger
Atenção, **cópias do bot** estão a circular **utilizando o nome RaidProtect**, convide-o apenas a partir do nosso site oficial.
:::

### Pré-requisitos {#requirement}

Para garantir o bom funcionamento do RaidProtect:
1. **Conceda a permissão de Administrador** ao RaidProtect. Isto permite-lhe moderar automaticamente todos os seus canais.
2. **Coloque o seu cargo ao mesmo nível dos administradores do servidor**. O sistema de permissões não permite que o RaidProtect sancione utilizadores com um cargo superior ao seu.
