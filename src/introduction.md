# Introduction

**RaidProtect** is an open-source Discord moderation bot written in Rust. It aims
to provide simple to use and powerful tools to moderate Discord servers and
secure them against common attacks.

This documentation describes the internal architecture of the bot and its features.
For a general overview see the README in the main repository.

[➔ RaidProtect repository (on GitHub)](https://github.com/raidprotect/raidprotect)

> **Note:** This documentation is intended for developers and not for end users.
>
> If you are looking for instructions to use the bot on your Discord server,
> see the [RaidProtect User Documentation](https://docs.raidprotect.org/).
