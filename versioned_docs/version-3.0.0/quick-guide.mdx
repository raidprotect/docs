Une fois les permissions correctement configurées (sans quoi le bot risque d'être inefficace), vous pouvez commencer à tester les différentes fonctionnalités ! 🤠 

## 💬 L'anti-spam {#anti-spam}

L'une des fonctionnalités principales de RaidProtect est bien évidemment son anti-spam. Ce dernier vous permet de **bloquer en une fraction de secondes les tentatives de spam**, sans aucune intervention humaine.

L'anti-spam prend en considération plusieurs paramètres pour définir la gravité du spam, et donc décider d'agir ou non. **Dans le cas où un spam est détecté, le bot expulse son auteur** et vous avertit dans le salon de logs de sa sanction. 🙌 

Dans certains cas, il peut être nécessaire d'**ignorer certains salons de la détection**. Pour ce faire, exécutez simplement la commande `?settings allowspam #salon-a-ignorer`. Pour plus de détails concernant le fonctionnement de l'anti-spam, consultez sa documentation détaillée !

## 💢 Le mode raid {#raidmode}

Parfois, un groupe d'utilisateurs menace votre serveur et il vous faut employer les grands moyens. Le mode raid est là pour ça ! Grâce à lui, vous pouvez **bloquer temporairement tous les utilisateurs tentant de rejoindre votre serveur**. Efficace, mais à réserver pour les grandes occasions… 🤩 

**L'activation du mode raid est d'une simplicité enfantine** - et cela vous permet de réagir rapidement face au danger. Faites tout simplement la commande `?raidmode` dans n'importe quel salon, et le mode raid sera en place. **N'oubliez pas de le désactiver** en utilisant la même commande !

Afin de rendre cette protection encore plus efficace, RaidProtect est capable de détecter un afflux massif de nouveaux utilisateurs et d'**activer le mode raid tout seul** (mais pas de le désactiver, attention). Si votre serveur est très populaire, il se peut que vous ayez besoin de changer la sensibilité de détection : référez-vous à la documentation détaillée pour en savoir plus.

## 🚧 Le captcha {#captcha}

Voici la dernière - mais pas moins populaire - fonctionnalité de RaidProtect. Vous connaissez sûrement les cases à cocher "Je ne suis pas un robot" ? Notre captcha a la même mission, mais sûr votre serveur Discord. Il permet **d'afficher un code à remplir à tous les nouveaux utilisateurs**, afin de stopper les robots se cachant sous un compte normal (_selfbot_). 🕵️ 

**Cette fonctionnalité est totalement optionnelle**, et il se peut que vous n'en ayez pas besoin. Si vous souhaitez tout de même l'utiliser, son activation se fait en utilisant la commande `?captcha enable`. Tout est ensuite automatique !

Si comme beaucoup de serveurs vous utilisez un rôle Membre et un bot permettant de l'assigner automatiquement (_autorole_), **cela risque d'interférer avec le captcha**. Heureusement, nous avons tout prévu. Consultez la documentation détaillée du captcha pour découvrir les paramètres optionnels.
